Introduction

Python is a versatile and powerful programming language known for its simplicity and readability. It is designed to be easy to understand and write, making it an excellent choice for both beginners and experienced developers. Python can be used for a wide range of applications, from web development to scientific computing, data analysis, artificial intelligence, and more.

Python comes with a comprehensive standard library that includes a wide variety of modules and functions to help you accomplish tasks efficiently. Some of the core modules include:

  • os: Provides functions for interacting with the operating system, such as file operations and directory manipulations.
  • math: Offers mathematical functions and constants for numerical calculations.
  • datetime: Allows manipulation of dates, times, and time intervals.
  • random: Provides functions for generating random numbers and making random selections.
  • json: Enables encoding and decoding JSON (JavaScript Object Notation) data.
  • urllib: Facilitates working with URLs and making network requests.

Python's ecosystem extends beyond the standard library, thanks to its rich package repository. The Python Package Index (PyPI) hosts thousands of third-party libraries that cater to various needs. These libraries cover areas such as web frameworks, data analysis, machine learning, and more. Some popular libraries include:

  • NumPy: A fundamental package for scientific computing with Python, providing support for arrays and mathematical functions.
  • Pandas: Offers data structures and tools for efficient data manipulation and analysis.
  • Django: A high-level web framework that simplifies the process of building robust and scalable web applications.
  • Flask: A lightweight web framework that allows you to quickly create web applications with minimal boilerplate code.
  • Matplotlib: A plotting library that enables the creation of various types of visualizations.

Whether you're a beginner looking to learn programming or an experienced developer working on complex projects, Python's versatility, community support, and vast ecosystem make it a valuable language to master.

Variables

In Python, variables serve as symbolic names for storing values within your code. These names, referred to as identifiers, adhere to specific rules and conventions.

A Python identifier must start with a letter (either uppercase or lowercase) or an underscore (_). Subsequent characters can also include digits (0-9). Python is case sensitive, distinguishing between uppercase and lowercase letters.

Unicode characters, including letters like "å" and "ü," can be used in Python identifiers. Unicode escape sequences can also be used as characters within identifiers. This allows for flexibility in naming variables.

It's important to choose meaningful and descriptive names for your variables to enhance the readability of your code. Variable names like total, count, or value can help others understand the purpose of the variable at a glance.

Python's emphasis on readability and adherence to naming conventions makes it easier to write and maintain code, especially when working on collaborative projects.

Declaring variables

In Python, you can declare variables using straightforward syntax. There are no explicit keywords like var or let as in JavaScript. Instead, you simply assign a value to a name, and Python automatically determines the scope and type of the variable.

Assigning a Value: You can declare a variable by directly assigning a value to it. Python will automatically create the variable and determine its scope based on where it's defined. For example:

x = 42

This creates a global variable x with a value of 42. See Variable scope below.

Variable scope

In Python, variable scope refers to where a variable is accessible and can be used within your code. There are two main types of variable scope: global scope and local scope.

  1. Global Scope: When you declare a variable outside of any function or code block, it becomes a global variable. Global variables are accessible from any part of your code, including functions and blocks. They maintain their value throughout the entire program. For example:
    2. global_var = 42 print(global_var)
  2. Local Scope: When you declare a variable within a function or a specific block, it becomes a local variable. Local variables are only accessible within the function or block where they are defined. They have a limited scope and are typically used for temporary storage within a specific context. For example:
    3. def my_function():
         local_var = 13
         print(local_var)
    my_function()

Unlike JavaScript's pre-ECMAScript 2015 behavior, Python's scoping rules are consistent across functions and blocks. Variables declared within a block, such as an if statement or a loop, maintain their scope only within that block. Similarly, Python doesn't allow you to access a variable declared inside a block from outside that block.

Constants

In Python, constants are typically represented using uppercase names to indicate that their values should not change during program execution. While Python does not have a direct const keyword like JavaScript, adhering to naming conventions helps achieve the same purpose.

Naming Conventions

  • Constants are often named using uppercase letters to distinguish them from variables.
  • By convention, constants' values remain unchanged throughout the program.
PI = 3.14

Behavior and limitations

  • Constants are not built-in to Python's syntax like in JavaScript; they rely on naming conventions.
  • Their values are not intended to be modified after assignment.

Scope and Re-declaration

  • Python does not enforce constant behavior through a specific keyword.
  • Variables declared with uppercase letters follow the convention of constants.

Even though Python doesn't have a dedicated const keyword, using uppercase letters and following naming conventions allows you to achieve similar functionality, promoting code clarity and understanding.

Control Structures

Control structures in Python are used to manage the flow of code execution based on conditions or to iterate over sequences of data. They help you make decisions, repeat actions, and control the logical flow of your program.

If statement

The if statement allows you to execute a block of code if a certain condition is met. You can also use elif (short for "else if") to add additional conditions, and else to provide a default action when no condition is met.

age = 18
if age < 18:
    print("You are a minor.")
elif age == 18:
    print("You just turned 18!")
else:
    print("You are an adult.")

While Loop

The while loop repeatedly executes a block of code as long as a given condition is true. This is useful for situations where you want to perform a task until a specific condition is no longer met.

count = 0
while count < 5:
    print("Count:", count)
    count += 1

For Loop

The for loop is used to iterate over a sequence (such as a list, tuple, string, or range) and execute a block of code for each item in the sequence.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print("I like", fruit)

Range Function

The range() function generates a sequence of numbers, which is often used with the for loop to iterate a specific number of times.

for i in range(5):
    print("Iteration:", i)

Data Types

In Python, data types define the nature of values that variables can hold. Python supports a variety of data types to accommodate different kinds of data. Here are some of the fundamental data types in Python

Numeric Data Types

  • int: Represents integer numbers. For example: 5, -10, 1000.
  • float: Represents floating-point numbers with decimal values. For example: 3.14, -0.5, 2.0.

Text Data Type

  • str: Represents strings of characters. Strings can be enclosed in single (' ') or double (" ") quotes. For example: "Hello", 'Python'.

Boolean Data Type

  • bool: Represents boolean values. It has two possible values: True and False. Used for logical operations and conditions.

Sequence Data Types

  • list: Represents ordered, mutable collections of elements. Lists can contain various data types. For example: [1, 2, 3], ['apple', 'banana', 'cherry'].
  • tuple: Represents ordered, immutable collections of elements. For example: (10, 20, 30).
  • set: Represents an unordered collection of unique elements. For example: {1, 2, 3}.

Mapping Data Type

  • dict: Represents a collection of key-value pairs. For example: {'name': 'John', 'age': 30}.

None Type

  • NoneType: Represents the absence of a value. Used as a placeholder or to indicate the absence of a return value in functions.
Data Structures

Data structures are essential tools in programming for organizing and storing data in a way that allows efficient manipulation and retrieval. Python provides a variety of built-in data structures that cater to different needs. Here are some of the most commonly used data structures in Python.

Lists

A list is a versatile data structure that can hold a collection of items. Lists are ordered and mutable, meaning you can add, remove, or modify elements after creation.

my_list = [1, 2, 3, 4, 5]
my_list.append(6)
my_list.remove(3)

Tuples

Similar to lists, tuples are ordered collections of items. However, tuples are immutable, meaning their contents cannot be changed after creation. Tuples are often used for data that should not be modified.

my_tuple = (10, 20, 30)

Sets

Sets are unordered collections of unique elements. They are useful for removing duplicates from a sequence and performing set operations like union, intersection, and difference.

my_set = {1, 2, 3, 4, 5}
my_set.add(6)

Dictionaries

Dictionaries store data as key-value pairs. Each key in a dictionary is unique, and you can use the key to retrieve its associated value quickly.

my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}
my_dict['occupation'] = 'Engineer'

Functions declaration

A function definition (also referred to as a function declaration or function statement) in Python consists of the def keyword, followed by:

  • The name of the function.
  • A list of parameters enclosed in parentheses and separated by commas.
  • A colon (:) to indicate the beginning of the function's code block.
  • The Python statements that define the function's behavior, indented within a block.

For instance, consider the following code that defines a simple function named square:

def square(number):
    return number * number

In this example, the function square takes a single argument called number. The function's behavior is defined by a single statement that returns the square of the given argument. The return statement specifies the value to be returned by the function.

return number * number

Primitive parameters (such as numbers) are passed to functions by value in Python. This means that the value is passed to the function, but any changes to the parameter's value within the function do not affect the original value outside the function or in the calling code.

Function definitions are essential for creating reusable pieces of code. They allow you to encapsulate logic, make your code modular, and promote code readability.
